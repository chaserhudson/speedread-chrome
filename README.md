# SpeedRead — Chrome Extension

RSVP (Rapid Serial Visual Presentation) speed reader for Chrome. Highlight any text on a webpage, press **Alt+S** or right-click and choose **Speed Read Selection**, and read one word at a time with the anchor letter highlighted in red.

## Features

- **ORP anchor highlighting** — The optimal recognition point of each word is highlighted in red and fixed at screen center, so your eyes never move
- **Adjustable speed** — 100 to 1000 WPM with a slider and presets (200, 300, 500)
- **Adaptive timing** — Short words fly by faster, long words linger, punctuation adds natural pauses
- **Keyboard shortcuts** — Space (play/pause), arrows (skip/speed), R (reset)
- **Scrubbable progress bar** — Click anywhere to jump
- **Context menu** — Right-click selected text to start reading
- **Fully offline** — No data ever leaves your browser

## Install from Source

1. Clone this repo
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select this folder

## Usage

1. Highlight text on any webpage
2. **Alt+S** or right-click → **Speed Read Selection**
3. **Space** to play/pause
4. **←/→** skip 10 words, **↑/↓** adjust speed, **R** reset
